Python PySpark Databricks Delta Lake SQL

Sobre o Projeto

Este projeto automatiza o monitoramento de preços de combustíveis coletados pela ANP (Agência Nacional do Petróleo). O sistema processa volumes massivos de dados diários para fornecer insights estratégicos sobre tendências de mercado e paridade de importação.

Arquitetura Medallion

B

Bronze Layer

Dados brutos ingeridos via API e arquivos CSV sem transformações, garantindo a linhagem histórica dos dados da ANP.

S

Silver Layer

Limpeza, tipagem de dados e filtragem de outliers. Aplicação de regras de negócio e enriquecimento geográfico.

G

Gold Layer

Tabelas agregadas prontas para consumo em Power BI, focadas em séries temporais e variações percentuais por região.
